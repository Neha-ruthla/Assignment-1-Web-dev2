# Assignment-1-Web-dev2
#  

##  Project Overview
This project is a web development assignment created using HTML, CSS, and JavaScript.  
It demonstrates basic front-end development concepts such as layout design, styling, and interactivity.

##  Objective
The objective of this assignment is to understand and implement core web development concepts including:
- Structuring web pages using HTML
- Styling using CSS
- Adding interactivity using JavaScript

##  Features
- Responsive webpage design  
- User-friendly interface  
- Interactive elements (buttons/forms/etc.)  
- Clean and structured layout  

##  Technologies Used
- HTML  
- CSS 
- JavaScript  
- VS Code  

##  Project Structure
- index.html → Main webpage  
- style.css → Styling file  
- script.js → JavaScript functionality  

##  How to Run
1. Download or clone the repository  
2. Open the project folder  
3. Double-click on `index.html`  
   OR  
   Open with Live Server in VS Code  


##  Learning Outcome
- Gained understanding of basic web development  
- Improved skills in UI design and layout  
- Learned how to connect HTML, CSS, and JavaScript
